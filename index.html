Atlanta Journal-Constitution — Next.js Migration & Performance

Context: Large media platforms with millions of monthly readers.

Problem

Legacy frontend architecture caused large bundle sizes and limited performance tuning.

What I Built

  • Led migration to Next.js 15 App Router across a core CMS monorepo.
  • Introduced React Suspense, lazy loading, and granular code splitting.
  • Standardized reusable frontend components adopted across multiple sites.

Outcome

  • Reduced JavaScript bundle size by ~80% (from ~1MB to ~150KB).
  • Improved performance and stability for revenue-critical, high-traffic pages.
  • Enabled faster feature delivery through shared frontend systems.

Tech

Next.js React TypeScript Webpack Server Components
